# Nitesh Kumar – Senior Frontend Portfolio (Next.js)

## Stack
- Next.js (App Router) + TypeScript
- TailwindCSS
- Framer Motion
- GSAP + ScrollTrigger
- React Three Fiber (subtle background depth)
- Lenis smooth scrolling

## Setup
```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
